# Cortex AI — Frontend Client

This directory contains the React SPA frontend application built with Vite, Tailwind CSS (v4), Redux Toolkit, Framer Motion, and Monaco Editor.

For full project setup instructions, environment variables configuration, and microservices architecture description, please refer to the **[Root README](../README.md)**.

## Local Development Quickstart

1. Ensure the root-level dependencies are installed.
2. Create `frontend/.env` using the template `frontend/.env.example`.
3. Start the dev server from this directory:
   ```bash
   npm run dev
   ```
   Or from the root directory:
   ```bash
   npm run dev:frontend
   ```
